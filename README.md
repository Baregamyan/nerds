# Nerds
Cайт дизайн студии из Краснодара.
## Разработка
1.Склонируйте проект: `git clone git@github.com:Baregamyan/nerds.git`

2.Установите зависимости:
```nodejs
npm install
```
3.Запустить локальный браузер и начать разработку.
```nodejs
npm run start
```

## Билд
1.Билд проекта
```nodejs
npm run build
```

## Деплой
1.Деплой проекта в **gh-pages**
```nodejs
npm run deploy
```
